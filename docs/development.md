## ⚙️ Development

### 🧪 Requirements

* Node.js 20+
* pnpm (recommended)

### 🔧 Setup

```bash
git clone https://github.com/YOUR_USERNAME/vitae.git
cd vitae
pnpm install
cp .env.example .env.local
```

Add your OpenAI API key to `.env.local`:

```env
OPENAI_API_KEY=sk-...
```

### 🚀 Run locally

```bash
pnpm dev
```
